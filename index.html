Pinyinification

This will convert Chinese(and Japanese!) characters to the exact same Chinese(or Japanese) characters with their pinyin(or Romanji(I learnt something new today)) next to them. So you can have the characters and the pinyin(or Romanji) right next to each other.

Is this useful? Who knows? It's good if you have like a bunch of characters spaced out and you want to have each individual pinyin(or Romanji) but all the characters are on the same line. So its only really good for specific cases. Google translate is probably better :(

This is a pretty "unrefined" use of pinyin-pro, and imports the pinyin-pro library which is actually in this html file. As this is a static file, i.e. its not going to be updated (It could be updated. But I'm not going to deal with Github Actions for something as silly as this). If you are curious, the pinyin-pro script comes from this version.

That means you do NOT need to be online to use this. If you have a (potentially offline) web browser you can use it(anything that can run javascript and html). There are probably bugs. I'm not gonna fix them. I got bored and I had an issue with getting pinyin into my obsidian so I made this.

Why can you use this offline? Well, this may be surprising, but this is actually all in one file! Yes, all the code to convert the pinyin is in this singular index file, which you can see on the github repo! You can just download the index.html file and uh yeah. Open it and enjoy.

It is probably very trivial to make different modes of pinyin conversion i.e. character to just pinyin, character to character then pinyin on the next line, etc. But I'm not going to. I'm lazy.
